# Visual Mode

focus on text
    - yv, c*
    - search
        - / (can use regex)
        - n
        - N
- visual mode
    - v\[motion\]
    - V\[motion\]
    - vw, vW, v$, v}, vj...
    - o: switch cursor direction
- caret mode
    - c in visual mode + \[motion\]
        - cw, cW, c$, c}, cj...
- y to copy
- p/P to search
