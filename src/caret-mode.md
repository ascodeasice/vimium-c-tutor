# Caret Mode

Caret Mode 提供除了用 `o` 以外，可以跳過一些文字，選取後方文字的方法。在 visual mode 按 `c` 可以進入 caret mode，在 caret mode 中，不會選取文字，只會移動游標位置。可以在確定游標的位置到了要複製文字的開頭，就可以按下 `v` 進入 visual mode 並複製文字。
