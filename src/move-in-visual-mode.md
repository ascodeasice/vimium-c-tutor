# 移動(Visual mode)

這些鍵只在 Visual Mode 中可以讓選取的字移動

| 按鍵 | 功能 |
| --- | ---- |
| h,j,k,l   | 左,下,上,右   |
| w   | 向後移動一個字   |
| W   | 向後移動一個字（忽略標點符號，只看空格）   |
| b   | 向前移動一個字   |
| e   | 移動到字尾   |
| $   | 移動到行尾   |
| o   | 切換鼠標的方向（朝前或朝後移動）   |
| y   | 複製目前選取的文字   |
| p   | 搜尋目前複製的文字   |
| P   | 搜尋目前複製的文字（新分頁）   |

- 大多數 vim motion 都有支援
